# DNS Shop Microservices

Учебный проект имитации магазина DNS на микросервисной архитектуре. Все компоненты собираются и запускаются внутри **Docker**.

## Структура проекта

*   **user-service** — управление пользователями (Java 21, Spring Boot 4.0.2, MapStruct, WebFlux, Liquibase, PostgreSQL 16, R2DBC).

### Эндпойнты User-Service


| Метод | Путь | Описание |
|-------|------|----------|
| `GET` | `/users` | Получить список всех пользователей, по фильтру |
| `GET` | `/users/{id}` | Получить пользователя по ID |
| `POST` | `/users` | Создать нового пользователя |

### Tестирование

Интеграциионные: настроен Testcontainers, jacoco = 86%

## Как запустить

Для запуска всей системы и базы данных выполните команду в корневой папке:


docker-compose up --build -d
